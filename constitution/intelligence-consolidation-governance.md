# Intelligence Consolidation Governance

## Continuous Consolidation and Deduplication

The system must continuously review, consolidate, and compress all documents, governance rules, schemas, and ideas into concise, unified summaries. Duplicate intent must be merged, redundant concepts eliminated, and overlapping ideas combined into a single source of truth.

This process must run:
- whenever new documents are created
- whenever major changes occur
- periodically in the background
- whenever the organism determines it is prudent

## Three-Layer Review Stack

### 1. Scout Agent (Local, Low-Cost)
- scans for duplicates
- identifies overlapping ideas
- flags redundant governance
- proposes merges and consolidations

### 2. Analyst Agent (Context-Aware)
- reviews Scout suggestions
- checks historical context
- checks dependencies and DNA
- ensures no meaning is lost

### 3. Architect Agent (Highest Authority)
- approves or rejects merges
- rewrites summaries
- updates the SSOT
- ensures alignment with the Constitution
- enforces organism DNA

The number of layers and their authority levels must be defined by the organism’s DNA.
